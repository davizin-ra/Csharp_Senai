# Projeto Passagem Aérea

Descrição do Projeto
O Sistema de Passagens Aéreas é um programa em C# que permite o cadastro e listagem de passagens para até dois passageiros. O sistema inclui autenticação via senha, um menu interativo para navegação e a exibição formatada das informações cadastradas.

Funcionalidades Principais:

#### 1. Autenticação de Usuário
- O sistema exige que o usuário insira uma senha válida ("123456") para acessar o menu.
#### 2. Cadastro de Passagens
- O usuário pode cadastrar até dois passageiros informando:
    - Nome
    - Origem
    - Destino
    - Data da viagem
    - Após o cadastro, o sistema permite cadastrar novos passageiros opcionalmente.
#### 3. Listagem de Passagens
- Exibição formatada dos bilhetes cadastrados, incluindo nome, origem, destino e data.
#### 4. Opção de Saída
- O usuário pode escolher sair do sistema a qualquer momento selecionando a opção apropriada no menu.
